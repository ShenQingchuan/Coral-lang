变量与常量

变量、常量来源于数学，是计算机语言中能储存计算结果或能表示值抽象概念。

变量可以通过变量名访问，Coral 语言变量名应为一个标识符。

声明变量、常量的一般形式是使用 var 或者 val 关键字：var 即 variables，val 即 value

常量与变量都可以没有初始化值，Coral 编译器会给他们一个默认的 “零值”。

编译器支持自动类型推断，即声明时可以不指定类型，由编译器来识别。

val a int = 1
val b = 1       // 系统自动推断变量类型为 int
val c int       // 如果不在声明时初始化，整型值的零值是 0
c = 1           // 对常量重新赋值会报错！

var x = 5       // 系统自动推断变量类型为Int
x += 1          // 变量可修改
