基础语法

Coral 程序可以由多个标记组成，可以是关键字，标识符，常量，字符串，符号。如以下语句由 6 个标记组成：

println("My name is " + name)

6 个标记是(每行一个)：

println
(
"My name is "
+
name
)

文件概念

包与模块

coral 强制要求程序员在第一行声明当前的包名。

package hello;

一来能够使得 Coral 根据文件路径组织代码模块， 二来这有助于解决导入时可能造成的 "名字空间" 冲突等问题。

如何建立起各个文件模块之间的关系呢？Coral 并不想让这件事情有太多学习成本， 以如下的项目文件结构为例：

app/
├── server.coral  (package app)
├── routes.coral  (package app)
├  --> 在这个文件里如果声明的 package 不是 app 会报错，包索引不会识别
│
├── controllers/  <-- app.controllers
│      └── UserController.coral   (package controllers)
│
├── services/     <-- app.services
│      ├── SignInService.coral    (package services)
│      └── SignUpService.coral    (package services)
│
└── models/       <-- app.models
       └── UserEntities/          <-- app.models.UserEntities
           ├── UserAccount.coral  (package UserEntities)
           └── UserProfile.coral  (package UserEntities)

这是一个基础的 Web REST 后端服务，执行时主入口是 server.coral，整个项目文件夹名为 app。

  • 首先解析最外层的根目录，再递归解析深层级的。

可以检索到 server.coral 和 routes.coral 两个文件模块， 和 controllers、services、models 三个子包。

所以在 server.coral 中你就可以这样引入路由模块：

import .routes;     // 用点号来表达相对路径

而在 routes.coral 中可能需要引入 controllers 中的模块：

import .controllers.UserController;

当然你也可以选择完整地引入一整个包，例如在注册帐号服务 SignUpService.coral 的代码中将引入用户相关的数据模型：

import app.models.UserEntities;

由于 SignUpService.coral 在很内部的位置，引入其他的包也就需要更长的索引链， 而且都是相对于项目文件夹根目录来索引的。

我们会添加一些好用的 IDE 的支持：

  • 帮助你在某文件夹创建新的源代码文件时，自动声明其包名为文件夹名，如 GoLand 一样方便。
  • 在某项目文件下书写代码时，使用到某个变量、函数或类名，会视情况自动导入

编译目标

Coral 的目标是将 .coral 源代码编译到 .cbytes 字节码文件，然后通过虚拟机执行。

我们倾向于让这门语言看上去像脚本语言一样，所以无需 main() 函数。

coral out.cbytes

终结分隔符

在 Coral 程序中，每个语句正如 C 家族中的其它语言一样以分号 ; 结尾。以下为两个语句：

var msg = "Hello World"
println(msg);

注释

注释不会被编译，甚至在词法分析阶段就被忽略了。

单行注释是最常见的注释形式，你可以在任何地方使用以 // 开头的单行注释。多行注释也叫块注释，均已以 /* 开头，并以 */ 结尾。如：

块注释编译时允许嵌套，但最多 5 层，具体可查看源代码 lexer 部分的实现。

// 单行注释
/*
 Author by 菜鸟教程
 我是多行注释
 */

标识符

标识符用来命名变量、类型等程序实体。一个标识符实际上就是 一个或是多个 Unicode 字符（没错，支持中文变量名）或下划线 _ 组成的序列， 但是第一个字符必须是 Unicode 字符或下划线，而不能是数字。

以下是有效的标识符：

msesh     圆周率   abc   move_name   a_123
myname50  _temp   j     a23b9       retVal

以下是无效的标识符：

  • 1ab 以数字开头
  • case Coral 语言的关键字
  • a+b 运算符是不允许的

转义字符

Coral 语言支持以下一些特殊的转义字符序列：

符号 字符含义
\a 响铃符
\b 退格
\t 制表符
\v 竖直方向制表符
\n 换行
\r 回车
\f 换页符
\" 双引号（字符串中）
\' 单引号（字符中）
\\ 反斜杠
\uXXXX 16进制 Unicode 字符
